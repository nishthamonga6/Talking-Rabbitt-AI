# 🐇 Rabbit AI – Talking Rabbitt MVP

Rabbit AI is a **Streamlit-based AI analytics platform** that allows users to upload datasets, visualize insights, explore data, and query data using a natural-language style assistant.

This project demonstrates a **conversational intelligence layer for business data**.

---

# 🚀 Features

### 📂 Dataset Manager
- Upload CSV datasets
- Store multiple datasets
- Preview uploaded data

### 📊 Analytics Dashboard
Generate interactive charts:
- Bar charts
- Line charts
- Scatter plots
- Histograms
- Box plots

### 🥧 Pie Chart Studio
Create pie charts separately by selecting:
- Category column
- Value column
